Force Simulator

Controls:

Force Buttons: Change the circles behavior.
Freeze: Stop and try something else.
Recipes: Some fun ideas and force explanations.
Lose your circles? Try scrolling down or adding Border Bounce and Seek Mouse

Recipes:

  • Multiply + Collide Bounce => Split some atoms. **Good place to start.
  • Gravity + Wrap => Really speed things up.
  • Freeze + Border Bounce + Seek Mouse => Gather your the circles together.
    • Once they're together add Collide Bounce for some extra action!
  • Align + Separate + Wrap + Cohesion => Circles will flock together.
    • Swap Cohesion with Seek Mouse to flock to the cursor
  • Seek Mouse + Eyes + Tickle + Collide Bounce => Weirdness
    • A huge thank you to all of the contributors to the p5.JS examples!
